## Overview

### Instalation

To install, just run:

```bash
$ npm install
```

### Development Watch

To start dev, just run:

```bash
$ npm start
```

With Docker & docker-compose:

```bash
$ make
```

---

### Compile Production

To compile for production, just run:

```bash
$ npm run build
```

With docker
