App estilo SPA para fins de aprendizado da tecnologia React.js

## About Idea

Um simples ToDo (lista de tarefas)

## Overview

### Instalation

Para instalar, basta rodar:

```bash
$ npm install
```

### Development Watch

Para instanciar a aplicação em desenvolvimento, basta rodar:

```bash
$ npm start
```
