## Overview

### Instalation

Para instalar, basta rodar:

```bash
$ npm install
```

### Development Watch

Para instanciar a aplicação em desenvolvimento, basta rodar:

```bash
$ npm start
```

---

### Compile Production

Para compilar a aplicação para produção, basta rodar:

```bash
$ npm run build
```
